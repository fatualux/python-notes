# Lists

![List](../static/images/lists.png)

In Python, **lists** are used to store multiple items in a single variable.

Lists are **ordered**, **mutable** and **allow duplicate values**.

```python
my_empty_list = []
my_empty_list = list() # same as above
my_list = [1, 2, 3]
my_list = list([1, 2, 3]) # same as above
[x + 7 for x in my_list] # it outputs [8, 9, 10]
list('dragon') # ['d', 'r', 'a', 'g', 'o', 'n']
```
