# Variables

In Python, a **variable** is a named <ins>storage <ins> location that <ins>holds data<ins>, which can be changed during the execution of the program.

![Variables](../static/images/variable.png)

Variables are used to store values for later use in the program, and are created when a value is assigned to them using the assignment operator (`=`).

Unlike in languages such as C, variables, in Python, do not need explicit declaration to reserve memory space: the interpreter automatically handles memory allocation and deallocation for them.

Example:
```python
x = 5
name = "Francesco"
my_list = [1, 2, 3]
my_tuple = (4, 5, 6)
my_dict = {"key1": "value1", "key2": "value2"}
```
