# Objects

In Python, an **object** is an abstraction for a data structure.

For example, a function can be an object, a class, a list can be an object; a variable can be an object too, and so on.

We can also affirm that <ins>an object is an **instance** of a class</ins>.

In this case, the word *instance* means an *example*, an *occurrence* of a class.
